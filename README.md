# flutter_shop

基于Flutter的商城项目

## 项目结构

```
lib/
├── api/              # API接口相关
├── assets/           # 资源文件
├── components/       # 通用组件
├── constants/        # 常量定义
├── main.dart         # 应用入口
├── pages/            # 页面文件
├── routes/           # 路由配置
├── stores/           # 状态管理
├── utils/            # 工具类
└── viewModels/       # 视图模型
```

### 各目录说明

- **api/**: 存放与后端交互的API接口定义和实现
- **assets/**: 存放图片、字体等资源文件
- **components/**: 可复用的UI组件
- **constants/**: 应用中使用的常量，如颜色、尺寸等
- **pages/**: 应用的各个页面
- **routes/**: 路由配置和导航相关
- **stores/**: 应用状态管理
- **utils/**: 通用工具函数
- **viewModels/**: 视图模型，连接UI和数据
